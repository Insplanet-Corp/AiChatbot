이력서 일괄 DB 등록 폴더
========================

이 폴더(resumes-inbox)에 이력서 파일을 넣고, 프로젝트 루트의
seed-resumes.bat 를 실행하면 폴더 안의 이력서를 하나씩 파싱해서
DB(resumes 테이블)에 자동으로 등록합니다.

[지원 형식]
  .pdf  .docx  .doc  .txt
  ※ 스캔본(이미지) PDF는 텍스트가 없어 이 스크립트로는 등록되지 않습니다.
     그런 파일은 앱(브라우저)에서 업로드하면 Vision OCR로 처리됩니다.

[실행 방법]
  1) 이 폴더에 이력서 파일들을 복사해 넣습니다.
  2) 프로젝트 루트의 seed-resumes.bat 을 더블클릭하거나,
     cmd 에서 다음을 실행합니다.
        seed-resumes.bat
  3) 등록에 성공한 파일은 자동으로 _done\ 폴더로 이동합니다.
     (재실행해도 _done\ 안의 파일은 다시 등록되지 않습니다)
  4) 실패한 파일은 이 폴더에 그대로 남습니다.
     원인을 확인해 고친 뒤 다시 실행하면 됩니다.

[다른 폴더 등록]
  seed-resumes.bat "D:\어딘가\이력서모음"

[테스트(저장 안 하고 파싱만)]
  seed-resumes.bat --dry-run

[주의]
  이 폴더와 _done\ 안의 파일에는 실제 개인정보(PII)가 평문으로 들어 있습니다.
  git 에 커밋되지 않도록 .gitignore 에 등록되어 있으니, 그대로 두세요.
